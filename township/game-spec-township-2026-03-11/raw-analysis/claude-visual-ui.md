# Claude Visual & UI Analysis — Township

## 1. Art Style

- **Visual Style**: Cartoon/cel-shaded isometric with 3D-rendered assets
- **Art Quality**: AAA mobile (Playrix production quality)
- **Consistency**: Highly consistent warm, rounded art style across all screens
- **Design Language**: Friendly, colorful, approachable — soft edges, exaggerated proportions
- Characters: Semi-realistic cartoon style (Rachel in frame 11, team avatars in frame 19)
- Buildings: Detailed isometric sprites with depth and lighting

## 2. Color Palette

### Primary Colors
- **Green**: #4CAF50 — grass, nature, "Send" buttons, positive actions
- **Blue**: #2196F3 — water, sky, UI panels, info elements
- **Yellow/Gold**: #FFC107 — coins, stars, highlights, rewards

### Secondary/Accent Colors
- **Orange/Red**: #FF5722 — notification badges, alert indicators, urgent timers
- **Purple**: #9C27B0 — premium content, special buildings (frame 22)
- **Brown**: #795548 — wood, earth tones, factory interiors

### Background
- Town view: Bright green grass, blue water, light sky gradient
- UI overlays: Dark semi-transparent backdrop (frames 3-8, 12-13)
- Match-3: Warm garden background with sunlight (frame 17)

### Color Psychology
- Warm, vibrant palette creates welcoming atmosphere
- Gold/yellow for rewards triggers dopamine association
- Green for positive actions (Claim, Send, Equip)
- Red badges for urgency/FOMO

## 3. UI Layout

### HUD Elements (Main Town View — frames 21-22)
- **Top-left**: Player level badge (circular, gold frame), XP progress
- **Top-center**: Coin counter (gold), product request bubbles
- **Top-right**: T-Cash counter, barn capacity, settings/menu
- **Left side**: Event timers stacked vertically (17D 6H, 6D 2H, 3D 6H, 6H 25M)
- **Right side**: Quick access buttons (store, map, events)
- **Bottom bar**: 5 main navigation icons (trophy, star/quests, cards, helicopter, construction)

### Screen Real Estate
- **Game area**: ~70% of screen (isometric town)
- **UI area**: ~30% (HUD + bottom bar + side buttons)
- **Modal overlays**: Center-screen with dark backdrop (factory, airport, pass)

### Menu Structure
- Bottom navigation bar with 5 primary sections
- Side panels for events/timers
- Modal popups for buildings, orders, events
- Full-screen overlays for Knight Pass, Cards

## 4. Typography

- **Title Font**: Bold, decorative, rounded serif — "YOUR CARDS", "WELL DONE!", "KNIGHT PASS"
  - All-caps with drop shadows, warm colors (gold/white)
  - 3D extruded appearance
- **Body Font**: Clean sans-serif for gameplay info, descriptions
- **Number Font**: Bold, clear digits for counters (coins, timers, levels)
- **Readability**: High — large text sizes, high contrast
- **Localization**: UI appears English-ready with adequate text space

## 5. Icons & Symbols

### Icon Style
- **3D rendered** with soft shadows and subtle gradients
- Consistent rounded corners and warm lighting
- Product icons: Realistic miniature food/goods illustrations

### Key Icons
- **Star** (gold, 5-point): Progression/completion currency
- **Coin** (gold circle): Soft currency
- **T-Cash** (green bill): Premium currency
- **Heart**: Social/friendship (frame 19)
- **Trophy**: Competitions/leaderboards
- **Barn** (red): Storage indicator

### Notification Patterns
- Red circle badges with white numbers (frames 7, 15, 19)
- Green checkmarks for completed/claimable items
- Timer overlays on event buttons (17D 6H format)

## 6. Animation & Effects (Inferred from Frames)

- **Confetti/particles**: Celebration effect in regatta screen (frame 5 — colorful dots)
- **Star animation**: Stars floating/landing in reward screen (frame 2)
- **Card reveal**: Card flip animation with sparkle (frames 8, 10 — unrevealed cards with diamond pattern)
- **Glow effects**: Green glow on card pack (frame 12), golden highlight on claimable rewards
- **Production smoke**: Steam/smoke from factory (frame 16 — pastry factory)
- **Fog/clouds**: Undiscovered area obscured by animated clouds (frame 9)

## 7. Screen Orientation & Aspect Ratio

- **Orientation**: Landscape (horizontal)
- **Aspect Ratio**: ~16:9 to ~21:9 (wide mobile screens)
- **Responsive**: UI elements anchored to edges, game view fills center
- **Device Target**: Phone primary (landscape), tablet compatible
- **Safe Area**: HUD positioned within safe margins, bottom bar respects home indicator area
