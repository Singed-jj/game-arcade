# Township MVP - Game Design Specification

## 1. Overview

| 항목 | 내용 |
|------|------|
| 장르 | 탑다운 2D 타운 빌더 + 농장 경영 |
| 엔진 | Godot 4.4 (GDScript) |
| 플랫폼 | Web (HTML5 export → Vercel) |
| 세션 길이 | 5-15분 |
| 아트 | 32x32 픽셀아트 타일 |
| 수익화 | 없음 (토이 프로젝트) |
| 레퍼런스 | Township (Playrix) — game-spec-township-2026-03-11/ 참조 |

## 2. Core Loop

```
농장에서 작물 심기
→ 시간 경과 후 수확
→ 공장에 투입
→ 가공품 생산
→ 헬리콥터 주문 납품
→ 코인/XP 획득
→ 레벨업
→ 새 건물 해금
→ 타운 확장
```

## 3. Map System

- **뷰**: 탑다운 2D 그리드
- **맵 크기**: 20x20 타일 (확장 가능)
- **타일 크기**: 32x32px
- **타일 종류**: 빈 땅(grass), 농장, 공장, 도로, 장식, 물
- **배치**: 건설 모드에서 드래그 앤 드롭으로 건물 배치
- **카메라**: 드래그로 패닝, 스크롤로 줌 (0.5x ~ 2.0x)

## 4. Building System

### 건물 데이터

| 건물 | 해금 레벨 | 비용 | 크기 | 기능 |
|------|----------|------|------|------|
| 밀 농장 | 1 | 50 | 2x2 | 밀 재배 (30초) |
| 옥수수 농장 | 3 | 120 | 2x2 | 옥수수 재배 (60초) |
| 당근 농장 | 5 | 200 | 2x2 | 당근 재배 (45초) |
| 제빵 공장 | 2 | 100 | 2x2 | 밀 → 빵 (60초), 슬롯 2개 |
| 통조림 공장 | 4 | 250 | 2x2 | 옥수수 → 통조림 (90초), 슬롯 2개 |
| 주스 공장 | 6 | 350 | 2x2 | 당근 → 주스 (75초), 슬롯 2개 |
| 헬리콥터 패드 | 1 | 0 (기본) | 3x3 | 주문 납품, 3개 주문 슬롯 |
| 창고 | 1 | 0 (기본) | 2x2 | 저장 용량 (기본 20, 확장 가능) |

### 건물 상태
- **농장**: idle → growing (타이머) → ready (수확 가능) → idle
- **공장**: idle → producing (타이머) → ready (수거 가능) → idle
- **헬리콥터 패드**: 항상 3개 주문 표시

## 5. Production Chain

### 레시피

| 입력 | 공장 | 출력 | 생산 시간 |
|------|------|------|----------|
| 밀 x1 | 제빵 공장 | 빵 x1 | 60초 |
| 옥수수 x1 | 통조림 공장 | 통조림 x1 | 90초 |
| 당근 x1 | 주스 공장 | 당근 주스 x1 | 75초 |

### 흐름
1. 농장 탭 → 작물 선택 (해당 농장에 맞는 작물만)
2. 타이머 진행 (농장 위에 프로그레스 바 표시)
3. 완료 시 "수확" 아이콘 표시 → 탭하면 창고에 저장
4. 공장 탭 → 생산 큐에 추가 (재료가 창고에 있어야 함)
5. 타이머 진행 → 완료 시 탭하면 창고에 저장

## 6. Order System (Helicopter)

- 3개 주문 슬롯이 항상 활성
- 각 주문: 1~3종 아이템, 각 1~3개
- 납품 조건: 모든 아이템이 창고에 있어야 "Send" 가능
- 보상: 코인 (아이템 가치 합계 x 1.5) + XP (아이템 수 x 10)
- 완료 → 즉시 새 주문 생성 (레벨에 맞는 난이도)
- 삭제: 주문 삭제 가능, 60초 쿨다운 후 새 주문

### 주문 난이도 (레벨 기반)
| 레벨 | 아이템 종류 | 개당 수량 |
|------|-----------|----------|
| 1-2 | 1종 | 1-2개 |
| 3-4 | 1-2종 | 1-2개 |
| 5-7 | 2-3종 | 1-3개 |
| 8-10 | 2-3종 | 2-3개 |

## 7. Storage (Warehouse)

- 모든 아이템은 창고에 저장 (원자재 + 가공품)
- 기본 용량: 20
- 확장: 50 코인당 +5 용량
- 용량 초과 시: 수확/생산 완료 불가 (경고 표시)
- 각 아이템은 1슬롯 차지 (밀 3개 = 3슬롯)

## 8. Leveling & Unlocks

| 레벨 | 필요 XP | 해금 |
|------|---------|------|
| 1 | 0 | 밀 농장, 헬리콥터, 창고 |
| 2 | 50 | 제빵 공장 |
| 3 | 120 | 옥수수 농장 |
| 4 | 200 | 통조림 공장 |
| 5 | 300 | 당근 농장 |
| 6 | 420 | 주스 공장 |
| 7 | 560 | 창고 확장 할인 |
| 8 | 720 | 농장 2번째 배치 가능 |
| 9 | 900 | 공장 슬롯 +1 |
| 10 | 1100 | MVP 엔딩 축하 |

## 9. UI Layout

```
┌─────────────────────────────────────────────┐
│ [Lv.3 ★] [🪙 1,250]           [📦 12/20]   │  HUD (top)
│                                              │
│                                              │
│          탑다운 타운 맵 (드래그 패닝)          │  Game area
│         건물 위에 프로그레스 바/아이콘          │
│                                              │
│                                              │
│ [🏗️ 건설] [🚁 주문] [📋 인벤토리]           │  Bottom bar
└─────────────────────────────────────────────┘
```

### 모달 화면
- **건설 모드**: 하단에 건물 목록 (가로 스크롤), 맵에 미리보기 배치
- **주문 패널**: 3개 주문 카드 목록, 각각 필요 아이템 + Send/Delete 버튼
- **인벤토리**: 그리드로 보유 아이템 표시
- **공장 UI**: 생산 슬롯 + 레시피 선택
- **농장 UI**: 작물 선택 (해금된 것만)

## 10. Save System

- **방식**: localStorage (Web), FileAccess fallback (Native)
- **SaveManager Autoload**: `scripts/managers/save_manager.gd`
- **저장 데이터**:
  - 건물 배치 (위치, 종류, 상태, 타이머 잔여)
  - 인벤토리 (아이템별 수량)
  - 플레이어 (레벨, XP, 코인)
  - 주문 (현재 3개 주문 내용)
- **저장 시점**: 상태 변경 시 즉시 (수확, 생산완료, 납품, 건설, 레벨업)

- 밀 농장 (idle, growing, ready) — 3 스프라이트
- 옥수수 농장 (idle, growing, ready) — 3 스프라이트
- 당근 농장 (idle, growing, ready) — 3 스프라이트
- 제빵 공장 (idle, producing) — 2 스프라이트
- 통조림 공장 (idle, producing) — 2 스프라이트
- 주스 공장 (idle, producing) — 2 스프라이트
- 헬리콥터 패드 — 1 스프라이트
- 창고 — 1 스프라이트
- 아이템 아이콘 (밀, 옥수수, 당근, 빵, 통조림, 주스) — 6개 (16x16)

- 코인 아이콘

- 버튼 배경 (건설, 주문, 인벤토리)
- 프로그레스 바 (타이머용)

### P2 (꾸미기)

## 13. Scope Boundary (Not Included)

MVP에 포함되지 않는 것:
- 매치3 미니게임
- 배틀패스/이벤트 시스템
- 카드 컬렉션
- 소셜/팀/채팅
- 사운드/음악
- 튜토리얼 (직관적 UI로 대체)
- 애니메이션/파티클 이펙트 (최소한만)
