요청 사항 요약

  • 🪨 이미지에 돌다리 20개 이상 존재

  • ✅ 5번째 돌다리부터 레벨 서클 배치
  • 📐 돌다리 크기에 맞게 서클 크기도 비례 조정
  • 💾 이미지 비율 기반 JSON으로 사전 정의
  • 📱 기기 viewport에 맞게 런타임 스케일링

JSON 데이터 구조 (안)

{
  "imageSize": { "w": 1080, "h": 1920 },
  "stones": [
    {
      "id": 1,
      "cx": 0.50,   // 중심 x / 이미지 w
      "cy": 0.08,   // 중심 y / 이미지 h
      "w":  0.18,   // 돌 width / 이미지 w
      "h":  0.06    // 돌 height / 이미지 h
    },
    ...
  ]
}

stones[0~3]: 스킵(장식용)
stones[4~]: level 1, 2, 3... 순서
