# Last War: Survival — SIDC Retention Loop Analysis

## Retention Loop Summary

| Phase | Strength | Key Element |
|-------|---------|-------------|
| Start | **Strong** | Immediate battle action + zombie narrative hook |
| Immersion | **Strong** | Satisfying army growth + visual spectacle in battles |
| Desire | **Strong** | Timed systems (Radar Tasks) + hero shard collection |
| Conversion | **Strong** | Multi-currency gating + gacha recruit system |

## Phase Details

### Start (첫 30초 훅)

| Element | Description | Evidence Frames |
|---------|-------------|-----------------|
| Instant Action | Highway battle likely first experience — auto-battle with immediate visual feedback | frame-011 |
| Zombie Narrative | "Please help me get rid of these zombies!" — clear, relatable story hook | frame-028 |
| Visual Spectacle | Even early battles show soldiers marching, vehicles, satisfying squad formation | frame-011, frame-014 |

The game hooks players within seconds by dropping them into an action sequence on a highway (frame-011). Auto-battle means zero learning curve. The zombie survival narrative provides emotional context (frame-028). Time-to-first-action is near-instant. The chibi-style blue helmet soldiers create visual identity from the first moment.

**Strength: Strong** — Minimal friction to first engagement. Auto-battle removes skill barriers.

### Immersion (몰입 유지)

| Element | Description | Evidence Frames |
|---------|-------------|-----------------|
| Army Growth | Visually dramatic scaling from ~10 to 100+ soldiers filling the screen | frame-014, frame-030, frame-033 |
| Battle Effects | Fire, ice, explosions, damage numbers create spectacle | frame-026, frame-029, frame-032 |
| Mode Variety | Story stages, Spec Ops, Frontline Breakthrough, Arena — prevents repetition | frame-018, frame-026, frame-017 |
| Short Sessions | ~30s battles (Spec Ops 3 = 32 seconds) maintain engagement without fatigue | frame-018 |
| Hero Management | Cooldown-based ability system adds decision-making layer | frame-029 |

The core immersion driver is visible army growth. Players start with ~10 soldiers (frame-014) and progress to massive armies filling the screen (frame-032, frame-033). Battle effects escalate — ice beams, fire rain, golden explosions, massive damage numbers (frame-026: -98, 38; frame-027: 8,158 x2).

Multiple battle modes prevent monotony: highway stages (frame-011), bridge Frontline (frame-026, frame-030), isometric boss battles (frame-027), timed Spec Ops (frame-018), Arena PvP (frame-017).

**Strength: Strong** — Army growth visualization is the killer feature. Multiple modes + short sessions = extended engagement.

### Desire (재방문 욕구)

| Element | Description | Evidence Frames |
|---------|-------------|-----------------|
| Radar Tasks | Timed restoration (6 tasks in 05:43:19) creates scheduled return triggers | frame-005 |
| Hero Shards | Incomplete collection drives "one more shard" return behavior | frame-009 |
| Alliance Social | Alliance chat, "Alliance War" breaking news create social obligation | frame-031 |
| Stage Progression | Always a next stage to beat | frame-014, frame-015, frame-029 |
| VIP System | Sustained engagement rewards | frame-031 |

Radar Task system (frame-005) is a classic timer-based return trigger — "6 Radar Task(s) will be restored in 05:43:19". Hero shard collection (frame-009: "Kimberly Shard, Owned: 0") drives completionist returns. Alliance features (frame-031) create social obligation.

**Strength: Strong** — Timer-based return triggers + social obligation + collection mechanics.

### Conversion (과금 전환)

| Element | Description | Evidence Frames |
|---------|-------------|-----------------|
| Multi-Currency | Gold tickets, red gems (57), diamonds, resources — complexity obscures value | frame-004, frame-017 |
| Arena Gating | 100 Diamonds for 1 challenge attempt — pay-to-compete | frame-017 |
| Gacha/Recruit | Card pack system with golden glow — excitement/gambling psychology | frame-004, frame-010 |
| Shard Gating | "Missing Items" -> "Obtain" button — direct IAP path from frustration | frame-009 |
| VIP Tiers | VIP 1 visible — tiered spending rewards | frame-031 |
| Mall/Shop | Dedicated shop interface with multiple tabs | frame-003 |

Multi-currency types (gold tickets, red gems, diamonds, resources) create a complex economy. Gacha/recruit system (frame-004, frame-010) uses card-reveal animation with golden glow. Arena gating (frame-017: 100 Diamonds per challenge, max 10 daily). Hero shard gating (frame-009: "Missing Items" with "Obtain" button). VIP system (frame-031) rewards sustained spending.

**Strength: Strong** — Multi-layered monetization with soft and hard gates. Every system has a spending path.

## Retention Risk Assessment

**Low Risk:**
- Core loop is extremely tight (build -> battle -> reward -> upgrade)
- Visual army growth provides continuous satisfaction
- Short battle sessions fit mobile usage patterns

**Medium Risk:**
- Heavy reliance on timers could frustrate impatient players
- Shard gating may create "paywall" perception
- Alliance dependency could be negative for solo players

**High Risk:**
- Multi-currency complexity may confuse new players
- Power gap between free and paying players could erode fairness
- Content exhaustion at endgame (Lv.120 with maxed systems)
