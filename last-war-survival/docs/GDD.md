# Last War Survival — Game Design Document

## 1. Overview
- **장르**: Army Clash / Crowd Runner (원작 Last War의 배틀 파트 재현)
- **핵심 컨셉**: 블루 헬멧 병사 군단을 이끌고 도로를 전진하며 좀비 적을 처치하는 자동 전투 + 히어로 스킬 게임
- **타겟 플랫폼**: Web (Godot 4 HTML5 export, Portrait 9:16)
- **세션 길이**: 30초~1분 (스테이지당)
- **수익화 모델**: 없음 (토이 프로젝트)

## 2. START — 첫 유저 경험

### 첫 화면
- 타이틀 "LAST WAR SURVIVAL" + 블루 헬멧 병사 일러스트
- 큰 "TAP TO START" 버튼 (pulsing 애니메이션)
- 배경: 도로가 위로 뻗어가는 원근감 연출

### 5초 내 시각적 흥미
- 타이틀 화면에서 바로 군단이 행진하는 미니 애니메이션
- 탭하면 즉시 Stage 1 진입 (로딩 없음)

### 30초 내 첫 성공 경험
- Stage 1은 3~4 웨이브의 약한 좀비 → 자동 전투로 쉽게 클리어
- "Mission Completed!" 배너 + 코인 보상

### Zero Friction Entry
- 터치 1회 → 게임 시작 (설정/계정 없음)
- 튜토리얼 없이 바로 플레이 (자동 전투이므로 즉시 이해 가능)

## 3. IMMERSION — 핵심 게임플레이

### 코어 루프
```
[군단 자동 전진] → [적 웨이브 등장] → [자동 전투 + 히어로 스킬 탭]
→ [적 처치 → 코인 획득] → [보스 등장 → 격파] → [Stage Clear → 보상]
→ [다음 스테이지] (반복)
```

### 핵심 메커니즘

**1. 군단 자동 전진**
- 블루 헬멧 병사들이 도로 위를 자동으로 위쪽으로 전진
- 적과 만나면 자동으로 공격 (근거리: 총, 원거리: 투사체)
- 병사 수가 곧 전투력 (시각적으로 군단 크기 = 강함)

**2. 히어로 스킬 시스템**
- 화면 하단에 히어로 카드 슬롯 3개
- 탭하면 스킬 발동 (쿨다운 7~10초)
- 스킬 종류:
  - **아이스 빔**: 전방 부채꼴 얼음 공격 (범위 데미지)
  - **에어스트라이크**: 지정 영역 폭격 (높은 데미지)
  - **힐링 웨이브**: 아군 체력 회복 + 일시 방어력 증가

**3. 적 웨이브 시스템**
- 각 스테이지 = 3~5 웨이브 + 보스
- 적 종류: 일반 좀비, 빠른 좀비, 탱크 좀비
- 보스: 대형 좀비 (화면 15% 크기), HP바 표시, 패턴 공격 (불비)

**4. 병사 모집**
- 도로 위에 "+5", "+10" 모집 게이트 등장
- 통과하면 병사 수 증가 → 군단이 시각적으로 성장
- 빼기 게이트도 존재 ("-3") → 회피 선택지

### 난이도 곡선 (Stepped)
| 스테이지 | 적 웨이브 | 적 HP | 보스 | 신규 메커니즘 |
|---------|----------|-------|------|-------------|
| 1-3 | 3 | 낮음 | 약함 | 기본 전투, 모집 게이트 |
| 4-7 | 4 | 중간 | 보통 | 빼기 게이트, 빠른 좀비 |
| 8-10 | 5 | 높음 | 강함 | 탱크 좀비, 보스 불비 패턴 |
| 11+ | 5 | 매우 높음 | 매우 강함 | 복합 패턴, 다중 보스 |

### 피드백 시스템
- **시각**: 데미지 숫자 팝업, 히트 시 적 깜빡임, 아이스빔/폭발 파티클
- **청각**: 공격 효과음, 보스 등장 경고음, Mission Complete 팡파레
- **성취감 리듬**: 30-45초마다 웨이브 클리어 → 코인 팝업 → 보스전 → 클리어 배너

## 4. DESIRE — 열망 유발

### Near-miss 설계
- 보스에게 거의 이길 뻔하다가 지면 → "SO CLOSE!" 메시지 + 남은 HP % 표시
- 코인을 모아 업그레이드하면 이길 수 있다는 시각적 힌트

### 스테이지 선택 화면
- 현재 스테이지 + 다음 스테이지 미리보기 (잠금 상태)
- 최고 기록 (처치 수) 표시 → 기록 갱신 욕구

### 업그레이드 시스템
- **군단 HP 업그레이드**: 코인으로 병사 기본 체력 증가
- **군단 공격력 업그레이드**: 코인으로 병사 기본 공격력 증가
- **시작 병사 수 업그레이드**: 코인으로 초기 병사 수 증가 (5 → 10 → 15...)
- 업그레이드 비용은 단계마다 증가 (100 → 200 → 400...)

### 파워 격차 인식
- 높은 스테이지에서 죽으면 → "Upgrade your army!" 버튼 표시
- 업그레이드 전/후 예상 전투력 비교 시각화

## 5. CONVERSION — 수익화 + 복귀

> 토이 프로젝트이므로 실제 과금 없음. 소프트 커런시(코인) 루프만 구현.

### Soft Currency Loop
- 코인 획득: 적 처치 (1~3), 웨이브 클리어 보너스 (10), 스테이지 클리어 보너스 (50)
- 코인 소비: 업그레이드 (100~수천), 계속하기 (스테이지 중간 부활, 50코인)
- 밸런스: 스테이지 1회 클리어 ≈ 업그레이드 1회 비용의 50~70%

### 복귀 훅
- 스테이지 선택 화면에서 "Best: Stage X" 표시 → 기록 경신 욕구
- 진행도 자동 저장 (localStorage)

### CONVERSION → IMMERSION 복귀
- 업그레이드 후 즉시 "PLAY" 버튼 → 바로 전투 재진입
- 부활 사용 시 즉시 전투 재개 (대기 없음)

## 6. Loop Integrity

```
[START]
  │ 탭 1회
  ▼
[IMMERSION: Stage Battle]
  │ 클리어 또는 실패
  ▼
[DESIRE: 업그레이드 / Near-miss]
  │ 코인 소비
  ▼
[CONVERSION: 업그레이드 적용]
  │ 즉시 복귀
  ▼
[IMMERSION: 다음 Stage] ← 루프 반복
```

**CONVERSION → IMMERSION 복귀 경로**:
- 업그레이드 화면에서 "PLAY" 탭 → 0.5초 내 전투 시작
- 중간 화면/로딩 없음
- 업그레이드 효과를 즉시 체감 (더 많은 병사, 더 높은 데미지)

## 7. Technical Spec

### 씬 구조 (예상)
```
Main (Node2D)
├── TitleScreen (Control)
├── StageSelect (Control)
├── BattleScene (Node2D)
│   ├── Road (Sprite2D / TileMap)
│   ├── PlayerArmy (Node2D)
│   │   └── Soldier (CharacterBody2D) x N
│   ├── EnemyManager (Node2D)
│   │   └── Enemy (CharacterBody2D) x N
│   ├── ProjectileManager (Node2D)
│   ├── EffectsLayer (CanvasLayer)
│   │   ├── DamageNumbers
│   │   ├── ParticleEffects
│   │   └── MissionBanner
│   └── BattleHUD (CanvasLayer)
│       ├── StageLabel
│       ├── HeroSlots (3x)
│       └── EnemyHPBar
├── UpgradeScreen (Control)
└── Managers (Autoload)
    ├── SaveManager
    ├── GameManager
    └── AudioManager
```
